# Logistack Ltd - Company Website

A modern, professional website for Logistack Ltd built with Next.js 14, Tailwind CSS, and Framer Motion.

## 🚀 Features

- **Next.js 14** with App Router for optimal performance and SEO
- **Tailwind CSS** with custom brand colors
- **Framer Motion** for smooth animations
- **Responsive design** that works on all devices
- **Contact form** with validation using React Hook Form and Zod
- **Multi-page architecture** with Home, Services, About, and Contact pages

## 🛠️ Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Forms:** React Hook Form with Zod validation
- **TypeScript:** Full type safety

## 🏃‍♂️ Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run development server:**
   ```bash
   npm run dev
   ```

3. **Open [http://localhost:3000](http://localhost:3000) in your browser**

4. **Build for production:**
   ```bash
   npm run build
   npm start
   ```
